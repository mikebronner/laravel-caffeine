# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 13.x    | ✅ |
| < 13.0  | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability, please email
[mike@genealabs.com](mailto:mike@genealabs.com) instead of opening a public
issue. All reports will be addressed promptly.
